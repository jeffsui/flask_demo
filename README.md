# flask_demo

flask demo

## flask project init

### 1. clone项目到本地

```shell
git clone https://github.com/jeffsui/flask_demo.git
```

### 2. 创建本地虚拟环境

```shell
python -m venv .venv
```

### 3. 激活虚拟环境

```powershell
 .\.venv\Scripts\Activate.ps1
```

### 4.安装依赖

```shell
pip install -r requirements.txt
```

### 5. 启动环境

```python
flask run 
```

### 6. 访问`http://localhost:5000`页面
